# FreeShot 产品需求文档 (PRD)

---

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | FreeShot |
| **产品定位** | 免费 macOS 录屏工具（屏幕录制 + 摄像头叠加）|
| **核心价值** | 完全免费、专为自媒体创作者打造 |
| **开源协议** | MIT |
| **目标用户** | 自媒体创作者、在线教育者、开发者演示 |

---

## 2. 核心功能

### 2.1 MVP（必须实现）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **区域录屏** | 选择屏幕区域进行录制 | P0 |
| **全屏录屏** | 录制整个屏幕 | P0 |
| **摄像头叠加** | 录屏时显示人脸画面（画中画）| P0 |
| **摄像头位置** | 拖拽调整摄像头位置（角落）| P0 |
| **麦克风录音** | 录制麦克风声音 | P0 |
| **系统声音** | 录制系统音频 | P0 |
| **录屏控制** | 开始/暂停/停止录制 | P0 |
| **视频保存** | 保存为 MP4 | P0 |
| **状态栏控制** | 菜单栏快捷操作 | P1 |

### 2.2 V1.0 功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **摄像头大小** | 调整摄像头画面尺寸 | P1 |
| **摄像头形状** | 圆形/方形/圆角 | P1 |
| **摄像头边框** | 边框颜色/粗细 | P2 |
| **鼠标点击显示** | 录制时显示点击效果 | P2 |
| **键盘显示** | 录制时显示按键 | P2 |
| **倒计时** | 3/5/10秒倒计时开始 | P2 |

### 2.3 V2.0 功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **视频剪辑** | 裁剪视频时长 | P2 |
| **GIF 导出** | 视频转 GIF | P3 |
| **云上传** | 生成链接分享（可选）| P3 |

---

## 3. 用户流程

### 3.1 录屏主流程
```
状态栏图标 → 点击 → 选择模式（区域/全屏）→ 开始录制 → 摄像头叠加显示 → 停止 → 保存
```

---

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术 |
|------|------|
| **UI** | SwiftUI + AppKit |
| **屏幕录制** | ScreenCaptureKit (SCStream) |
| **摄像头** | AVFoundation (AVCaptureSession) |
| **视频合成** | AVAssetWriter |
| **音频** | AVAudioEngine |
| **打包** | Xcode + XcodeGen |
