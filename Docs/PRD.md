## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | FreeShot |
| **一句话定位** | 免费开源的 macOS 截图与录屏工具，专为内容创作者打造 |
| **核心价值** | 零成本替代 CleanShot X / ScreenFlow，开箱即用 |
| **开源协议** | MIT |
| **最低系统** | macOS 13.0 (Ventura) |
| **技术栈** | Swift 5.9 · SwiftUI + AppKit · ScreenCaptureKit · AVFoundation |
| **构建工具** | XcodeGen → Xcode 16 |

## 3. 用户流程

### 3.1 截图流程

```
菜单栏图标 → 点击 → Screenshot 标签
 ├─ Capture Region  → 拖拽选区 → 松手 → 预览浮窗 → 保存/复制/标注/OCR
 ├─ Capture Window  → 点击窗口 → 预览浮窗 → 保存/复制
 ├─ Capture Full Screen → 立即截取 → 预览浮窗 → 保存/复制
 ├─ Scroll Capture  → 开始滚动 → 自动捕获 → 停止滚动/ESC → 保存长图
 └─ Multi-Display   → 显示器列表 → 选择 → 截取 → 保存
```

### 3.2 录屏流程

```
菜单栏图标 → 点击 → Recording 标签
 ├─ 配置选项：摄像头(开关/位置/大小) · 倒计时 · 鼠标点击 · 键盘显示 · 音频
 ├─ Region 录制 → 拖拽选区 → 倒计时 → 开始录制
 └─ Full Screen 录制 → 倒计时 → 开始录制
     │
     ├─ 录制中面板：显示时长 · 暂停/恢复按钮 · 停止按钮
     ├─ 暂停 → 帧停止写入 → 恢复 → 时间戳偏移连续
     └─ 停止 → finishWriting → 文件验证 → NSSavePanel → 保存 MP4
```

### 3.3 权限流程

```
首次启动
 ├─ 屏幕录制 → CGRequestScreenCaptureAccess() → 系统弹窗
 ├─ 摄像头   → AVCaptureDevice.requestAccess(.video) → 系统弹窗
 └─ 麦克风   → AVCaptureDevice.requestAccess(.audio) → 系统弹窗

录制/截图时如权限被拒
 └─ NSAlert 提示具体缺少哪个权限 + 引导到系统设置路径
```

### 3.4 后处理流程

```
录屏完成后
 ├─ 视频裁剪 → 选择文件 → 输入起止时间 → 导出 MP4
 ├─ GIF 导出 → 选择文件 → 导出 GIF
 └─ 分享     → NSSharingServicePicker 系统分享 / 本地文件夹
```

## 5. 技术架构

### 5.1 模块依赖

```
PopoverView (SwiftUI)
 ├── ScreenshotTabView
 │    ├── ScreenshotManager        ← CGWindowListCreateImage
 │    ├── AutoScrollCaptureManager ← Timer + NSEvent scroll monitor
 │    ├── MultiDisplayManager      ← SCShareableContent displays
 │    └── OCRManager               ← Vision VNRecognizeTextRequest
 ├── RecordingTabView
 │    └── RecordingManager         ← SCStream + AVAssetWriter
 │         ├── CountdownWindow
 │         ├── CameraPreviewWindow ← AVCaptureSession
 │         ├── MouseClickOverlay   ← CGEvent tap
 │         └── KeystrokeOverlay    ← CGEvent tap
 ├── SettingsView                  ← @AppStorage
 └── ScreenshotHistoryView         ← ScreenshotHistory
```

### 5.2 关键技术决策

| 决策 | 方案 | 原因 |
|------|------|------|
| 屏幕录制 API | ScreenCaptureKit (非 AVCaptureScreenInput) | macOS 13+ 推荐，支持 sourceRect 区域录制 |
| 视频编码 | AVAssetWriter (H.264 + AAC) | 输出标准 MP4，兼容性最好 |
| 暂停实现 | isPaused flag + CMSampleBufferCreateCopyWithNewTiming | 跳帧 + 时间戳偏移，恢复后无跳跃 |
| 坐标系统 | 统一转换为 CG 坐标 (top-left origin) | AppKit 是 bottom-left，SCK 要求 top-left |
| App 形态 | LSUIElement 菜单栏 App | 无 Dock 图标，不占任务栏 |
| 项目管理 | XcodeGen (project.yml) | 避免 .xcodeproj 合并冲突 |

### 5.3 权限需求 (TCC)

| 权限 | 用途 | 触发时机 |
|------|------|----------|
| 屏幕录制 (Screen Recording) | 截图 + 录屏 | App 启动时预请求 |
| 摄像头 (Camera) | 画中画 | 开启摄像头时请求 |
| 麦克风 (Microphone) | 录制麦克风音频 | 开启麦克风时请求 |
| 辅助功能 (Accessibility) | 键盘按键监听 | 开启键盘显示时请求 |

## 6. 非功能需求

| 维度 | 要求 |
|------|------|
| **启动时间** | 冷启动 < 2s |
| **录屏帧率** | 60fps (SCStreamConfiguration) |
| **视频码率** | 默认 10 Mbps (H.264 High Profile) |
| **音频码率** | 128 kbps AAC, 44.1kHz, 双声道 |
| **内存占用** | 截图待机 < 30MB；录屏中 < 150MB |
| **最大录制时长** | 无硬限制（受磁盘空间约束） |
| **Retina 支持** | 截图/录屏按物理像素 (backingScaleFactor) |
| **多显示器** | 支持列出并选择任意显示器 |
| **文件格式** | 截图: PNG/JPEG/TIFF；录屏: MP4 (H.264+AAC) |

## 7. 边界与约束

### 7.1 不做的事 (Out of Scope)

- 直播推流 (RTMP/RTSP)
- 视频特效/滤镜/转场
- 多轨时间线编辑
- Windows/Linux 跨平台
- 云存储服务端 (仅本地分享)
- 虚拟摄像头输出
- 画中画 compositing 到视频流中（当前为独立浮窗叠加）

### 7.2 已知限制

| 限制 | 说明 |
|------|------|
| macOS 版本 | 仅支持 macOS 13.0+（ScreenCaptureKit 要求） |
| 摄像头合成 | 摄像头画面为独立浮窗，不合成到视频流中 |
| 系统音频 | 依赖 ScreenCaptureKit capturesAudio，部分 App 可能静音 |
| 快捷键 | 固定快捷键，暂不支持自定义 |
| 签名 | 开发者签名，非 App Store 分发需 Gatekeeper 放行 |
