## 下一步测试

> 以下项目需要在 GUI 中授予屏幕录制权限后手动验证

1. **录屏 E2E**：全屏录制 → 停止 → 检查 MP4 有画面有声音
2. **区域录屏精度**：选区与实际录制区域是否匹配
3. **暂停/恢复**：暂停后帧是否停止，恢复后继续
4. **连续录制**：录制→停止→录制→停止，无崩溃
5. **截图全流程**：截图→预览→保存/复制/标注/OCR
6. **快捷键**：⌘⇧4/5/6/R/F/S 是否正常触发
7. **Retina 缩放**：高分辨率下选区坐标是否准确
