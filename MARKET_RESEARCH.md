# ScreenCam Pro 市场调研报告

---

## 一、产品定位

| 项目 | 内容 |
|------|------|
| **产品名称** | ScreenCam Pro（暂定）|
| **产品定位** | macOS 录屏工具（屏幕录制 + 摄像头叠加）|
| **核心功能** | 录屏时同时显示摄像头画面 |
| **目标用户** | 开发者做演示、在线教育、内容创作者 |

---

## 二、竞品功能对比

### CleanShot X 录屏功能

| 功能 | 描述 |
|------|------|
| **屏幕录制** | 录制整个屏幕/区域 |
| **摄像头叠加** | 录屏时显示人脸画面（可拖拽位置）|
| **麦克风录音** | 支持系统声音 + 麦克风 |
| **鼠标点击显示** | 录制时显示鼠标点击效果 |
| **键盘输入显示** | 录制时显示按键 |
| **摄像头效果** | 背景虚化、调整大小/形状 |
| **GIF 导出** | 视频转 GIF |
| **内置剪辑** | 裁剪、剪辑视频 |

### 其他竞品

| 产品 | 特点 |
|------|------|
| **OBS Studio** | 免费开源，强大但复杂 |
| **ScreenFlow** | 专业级，价格高 $169 |
| **Camtasia** | 付费 $249，功能全 |
| **Loom** | 在线录制，依赖网络 |

---

## 三、技术可行性

### 核心功能实现方案

| 功能 | 技术方案 | 难度 |
|------|----------|------|
| **屏幕录制** | ScreenCaptureKit (SCStream) | ⭐⭐ |
| **摄像头捕获** | AVFoundation (AVCaptureSession) | ⭐⭐ |
| **画面合成** | CALayer 叠加 + AVAssetWriter | ⭐⭐⭐ |
| **音频混合** | AVAudioEngine 混音 | ⭐⭐ |
| **鼠标/键盘** | CGEventtap 监听 | ⭐ |
| **视频导出** | AVAssetWriter | ⭐⭐ |
| **GIF 转换** | ImageIO | ⭐⭐ |

### macOS API 支持

- **ScreenCaptureKit** (macOS 12.3+) - 屏幕录制
- **AVFoundation** - 摄像头捕获
- **AVAudioEngine** - 音频处理
- **Core Graphics** - 视频编码

## 五、结论

### ✅ 可行性确认

1. **技术可行**：ScreenCaptureKit + AVFoundation 可实现所有核心功能
2. **市场存在**：CleanShot/ScreenFlow 验证需求
3. **差异化**：开源 + 免费 + 本土化

### 📋 建议

- **MVP**：录屏 + 摄像头叠加 → 验证核心价值
- **V1.0**：音效、效果增强
- **V2.0**：剪辑、云端

---

**调研结论：项目可行，建议推进。**
