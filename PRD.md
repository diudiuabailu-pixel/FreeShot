## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | FreeShot |
| **一句话定位** | 免费开源的 macOS 截图 & 录屏工具，专为自媒体创作者打造 |
| **核心价值** | 截图录屏一体化 + 标注/OCR/GIF 后处理 + 完全免费无广告 |
| **开源协议** | MIT |
| **最低系统** | macOS 13.0 (Ventura)+ |
| **技术栈** | Swift 5.9 + SwiftUI + AppKit + ScreenCaptureKit |

### 1.1 为什么做 FreeShot？

macOS 截图录屏工具市场现状：
- **CleanShot X**：功能强大但 $29 付费
- **Snagit**：企业级但 $63 昂贵
- **iShot**：国内免费但功能有限，有广告
- **系统自带**：基础功能，无标注/OCR/GIF

**FreeShot 的差异化**：
- ✅ **完全免费 + 开源**（MIT）
- ✅ 截图 + 录屏 + 后处理一体化
- ✅ 原生 Swift，性能优于 Electron 方案
- ✅ 面向自媒体创作者的工作流优化

## 2. 目标用户

### 2.1 核心用户画像

| 用户类型 | 典型场景 | 痛点 |
|---------|---------|------|
| **自媒体创作者** | 教程录制、产品演示、短视频制作 | 需要录屏+人脸出镜，不想付费买 CleanShot |
| **开发者** | Bug 截图、代码演示、技术分享 | 需要快速截图+标注+OCR |
| **产品经理** | 需求标注、竞品截图、原型录屏 | 需要截图标注后快速分享 |
| **在线教育** | 课程录制、操作演示 | 需要摄像头画中画+键盘显示 |

### 2.2 使用场景优先级

1. **高频**：区域截图 → 快速标注 → 复制/保存
2. **高频**：全屏截图 → OCR 取文字
3. **中频**：录屏 → 摄像头叠加 → 保存 MP4
4. **中频**：截图 → 历史查看 → 再次使用
5. **低频**：录屏 → 裁剪 → 转 GIF → 分享

## 5. 用户流程

### 5.1 截图流程
```
菜单栏图标 → 选择截图模式 → 截取 → 预览浮窗 → 复制/保存/标注/OCR
                                                    ↓
                                              自动存入历史
```

### 5.2 录屏流程
```
菜单栏图标 → 选择录屏模式 → 设置选项（摄像头/音频/倒计时）
    ↓
倒计时 → 开始录制 → 录制指示器显示
    ↓
暂停/恢复 → 停止 → 保存面板 → 选择路径 → 保存 MP4
```

### 5.3 后处理流程
```
截图 → 标注 → 保存标注图
截图 → OCR → 文字复制到剪贴板
录屏 → 视频裁剪 → 保存裁剪后视频
录屏 → GIF 导出 → 保存 GIF
```

## 6. 技术架构

### 6.1 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| UI 框架 | SwiftUI + AppKit | 界面渲染 |
| 屏幕捕获 | ScreenCaptureKit (SCStream) | 截图 & 录屏 |
| 视频写入 | AVAssetWriter | MP4 编码 |
| 摄像头 | AVCaptureSession | 画中画 |
| 文字识别 | Vision (VNRecognizeTextRequest) | OCR |
| 图像处理 | Core Graphics + ImageIO | 标注 & GIF |
| 视频编辑 | AVAssetExportSession | 裁剪 |

### 6.3 关键技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 屏幕捕获 | ScreenCaptureKit | macOS 13+ 官方推荐，替代废弃的 CGWindowList |
| 视频编码 | H.264 via AVAssetWriter | 兼容性最好，硬件加速 |
| 坐标系统 | 统一转 CG 坐标 | 避免 AppKit/CG 坐标混用导致区域偏移 |
| 暂停机制 | 帧级跳过 | 在 SCStreamOutput 中 guard isPaused |
| 时间戳对齐 | 首帧 PTS 启动 | 避免 startSession(.zero) 导致黑屏 |

## 11. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| macOS 权限变更 | 新版系统可能收紧 ScreenCaptureKit 权限 | 跟进 WWDC，及时适配 |
| 快捷键冲突 | 与其他 App 快捷键冲突 | 后续增加自定义快捷键设置 |
| Retina 坐标偏差 | 高缩放下选区不准 | 已统一 CG 坐标 + scaleFactor |
| 连续录制稳定性 | 长时间录制可能内存泄漏 | resetState() + weak self |
