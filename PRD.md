## 2. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | FreeShot |
| **一句话定位** | 免费开源的 macOS 截图 & 录屏工具，专为自媒体创作者打造 |
| **核心价值** | 截图录屏一体化 + 标注/OCR/GIF 后处理 + 完全免费无广告 |
| **开源协议** | MIT |
| **最低系统** | macOS 13.0 (Ventura)+ |
| **技术栈** | Swift 5.9 + SwiftUI + AppKit + ScreenCaptureKit |

### 2.1 产品愿景

FreeShot 的目标不是做一个"功能看起来很多、但真实使用容易翻车"的演示项目，而是做一个**用户可以在真实 macOS 环境下完成截图、录屏、导出与轻量后处理主流程**的工具。

> **先把截图 / 录屏主链路做稳，再谈扩展与外围能力。**

### 2.2 为什么做 FreeShot？

macOS 截图录屏工具市场现状：
- **CleanShot X**：功能强大但 $29 付费
- **Snagit**：企业级但 $63 昂贵
- **iShot**：国内免费但功能有限，有广告
- **Shottr**：截图好用但无录屏
- **Kap**：开源录屏但无截图
- **系统自带**：基础功能，无标注/OCR/GIF

**FreeShot 的差异化组合价值**：
1. **完全免费 + 开源**（MIT）
2. 截图 + 录屏 + 后处理一体化
3. 原生 Swift，性能优于 Electron 方案
4. 本地优先，核心功能不依赖云服务
5. 中文语境友好，适合本地创作者

## 3. 目标用户

### 3.1 用户画像

| 用户类型 | 典型场景 | 痛点 |
|---------|---------|------|
| **自媒体创作者** | 教程录制、产品演示、短视频制作 | 需要录屏+人脸出镜，不想付费 |
| **开发者** | Bug 截图、代码演示、技术分享 | 需要快速截图+标注+OCR |
| **产品经理** | 需求标注、竞品截图、原型录屏 | 需要截图标注后快速分享 |
| **在线教育** | 课程录制、操作演示 | 需要摄像头画中画+键盘显示 |

### 3.2 使用场景优先级

1. **高频**：区域截图 → 快速标注 → 复制/保存
2. **高频**：全屏截图 → OCR 取文字
3. **中频**：录屏 → 摄像头叠加 → 保存 MP4
4. **中频**：截图 → 历史查看 → 再次使用
5. **低频**：录屏 → 裁剪 → 转 GIF → 分享

## 8. 用户主流程

### 8.1 截图主流程
```
状态栏图标 / 快捷键 → 选择模式（区域/窗口/全屏）→ 截取 → 预览浮窗 → 复制/保存/标注/OCR
                                                                         ↓
                                                                   自动存入历史
```

### 8.2 录屏主流程
```
状态栏图标 / 快捷键 → 选择模式（区域/全屏）→ 配置选项 → 倒计时 → 开始录制
    ↓
录制指示器 → 暂停/恢复 → 停止 → 保存面板 → MP4
```

### 8.3 当前阶段最低可接受体验

用户至少必须稳定完成：
1. 发起截图
2. 发起录屏
3. 正常停止录屏
4. 获取可播放且内容正确的文件
5. 连续多次使用不崩溃

**如果以上路径不能稳定达成，则不能视为可发布。**

### 9.3 关键技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 屏幕捕获 | ScreenCaptureKit | macOS 13+ 官方推荐，替代废弃的 CGWindowList |
| 视频编码 | H.264 via AVAssetWriter | 兼容性最好，硬件加速 |
| 坐标系统 | 统一转 CG 坐标 | 避免 AppKit/CG 坐标混用导致区域偏移 |
| 暂停机制 | 帧级跳过 | 在 SCStreamOutput 中 guard isPaused |
| 时间戳对齐 | 首帧 PTS 启动 | 避免 startSession(.zero) 导致黑屏 |

## 13. 验收标准（量化）

### 13.1 截图验收

- 区域/窗口/全屏截图连续 **10 次以上无崩溃**
- 截图内容与选择区域一致，无明显偏移
- 预览、保存、复制、历史至少完成一次真机验证
- 标注保存结果正确

### 13.2 录屏验收

- 区域录屏与全屏录屏在真机上完成 **10 次连续录制无崩溃**
- 停止后生成的 MP4 可正常播放
- 不允许高频出现：黑屏、空文件、时长异常、无法生成文件
- 暂停/恢复真实作用于录制结果
- 重复录制不因状态未清理而崩溃

### 13.3 多显示器 / Retina 验收

- 至少 2 组显示环境验证：单屏 Retina + 双屏/外接
- 选区与录制/截图结果一致
- 不允许坐标翻转、整体偏移、尺寸错误

### 13.4 权限验收

- 首次失败时给出明确权限提示
- 用户能明确知道去哪里开启
- 未授权时不能无提示失败
